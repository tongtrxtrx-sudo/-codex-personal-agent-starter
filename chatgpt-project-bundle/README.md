# ChatGPT Project 上传包

这个目录用于把当前本地智能体的核心设定同步到 ChatGPT Project 中，作为“驾驶舱版本”的上下文。

## 适用场景

- 在网页端 ChatGPT Project 中使用
- 在手机 ChatGPT app 中查看、提问、继续推进任务
- 需要长期保存你的工作方式、当前重点和常用提示词

## 不适用场景

这个上传包不是本地执行层，不能替代 Codex 在本地仓库里的这些能力：

- 自动发现 `.agents/skills/`
- 执行本地脚本
- 访问你的 `D:` 盘文件
- 直接运行命令和改本地代码

## 建议上传方式

优先上传这 4 个文件：

1. `01-智能体身份与边界.md`
2. `02-默认工作方式.md`
3. `03-当前重点.md`
4. `04-常用提示词.md`

## 推荐用法

- 在 ChatGPT Project 里把这些文件作为长期上下文
- 让 ChatGPT 帮你做规划、总结、简报、需求梳理和任务拆解
- 真正需要读写文件、改代码、跑命令时，再回到本地 Codex 仓库执行

## 一句话分工

ChatGPT Project 负责长期驾驶舱，Codex 仓库负责本地执行层。
