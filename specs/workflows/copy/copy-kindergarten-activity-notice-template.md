# 幼儿园活动通知模板

## 目标
用于同步班级活动的时间、地点、着装、准备事项和家长配合要求。

## 适用场景
- 亲子活动通知
- 秋游 / 冬游通知
- 讲座通知
- 升旗或主题活动通知

## 语料锚点
- 活动具体安排如下
- 活动准备：统一穿...
- 活动时间 / 活动地点 / 接送地点
- 活动仅限 1 位家长陪同
- 谢谢大家的支持与配合

## 最少需要的信息
- 活动名称
- 活动时间
- 活动地点
- 参加对象
- 穿着或物品要求
- 接送安排

## 默认结构
1. 开场称呼
2. 活动目的或一句背景
3. 分点列出时间、地点、准备事项
4. 补充陪同或接送规则
5. 温和收尾

## 高频表达锚点
- `亲爱的家长朋友们：`
- `具体安排如下`
- `活动准备`
- `活动仅限 1 位家长陪同`
- `谢谢大家的支持与配合`

## 输出骨架

```md
各位家长朋友们：

本周我们将开展秋游活动，具体安排如下😊

1⃣️活动时间：本周五上午 8:30 - 10:30  
2⃣️活动地点：柔石公园  
3⃣️活动准备：统一穿园服、舒适运动鞋，带水杯、采集篮、适量小零食和垃圾袋  
4⃣️接送安排：8:00 - 8:30 在北门集合，10:30 - 11:00 接回

请大家提前帮孩子准备好物品，谢谢大家的支持与配合🌷
```

## 审校重点
- 时间、地点、接送安排必须齐全
- 着装和物品要求要短句清楚
- 活动氛围要轻松，但不能漏掉硬信息

## 待确认项
- 是否允许家长陪同
- 是否需要 `@所有人`
- 是否有天气或安全补充说明
