# 幼儿园返园提醒模板

## 目标
用于周末、节假日或短休后提醒孩子返园，重点是调整作息、说明时间、提示需带物品。

## 适用场景
- 节后返园提醒
- 周末后返园提醒
- 开学前一晚提醒
- 返园当天早晨提醒

## 语料锚点
- 今晚宝贝们早些休息
- 我们明天幼儿园不见不散哦
- 老师在幼儿园里等你们回来
- 请给宝贝们检查好指甲
- 带上水杯准备开启快乐的幼儿园生活

## 最少需要的信息
- 返园日期
- 是否有具体入园时间
- 需带物品
- 是否有园服、床品、作业等特殊要求
- 是否需要提醒接送家人

## 默认结构
1. 开场称呼
2. 返园时间提醒
3. 作息调整提醒
4. 物品或动作要求
5. 温和收尾

## 高频表达锚点
- `爸爸妈妈们：`
- `假期已接近尾声`
- `今晚宝贝们早些休息`
- `老师在幼儿园里等你们回来`
- `我们明天不见不散哦`

## 输出骨架

```md
爸爸妈妈们：

假期就要结束啦，宝贝们明天要返园啦😊 今晚记得让孩子们早些休息，慢慢把作息调整回来哦。

明天入园时间为 8:00 - 8:30，请帮宝贝准备好小书包、水杯、纸巾或湿巾和换洗衣物。也记得帮孩子检查一下指甲，如有特殊情况欢迎提前和老师沟通❤️

老师在幼儿园里等你们回来，我们明天不见不散哦🌷
```

## 审校重点
- 返园时间和动作要求明确
- 收假氛围温和，不制造焦虑
- 如果涉及老人接送，要提醒交接

## 待确认项
- 具体入园和离园时间
- 是否有园服、床品、作业要求
- 是否保留 `@所有人`
