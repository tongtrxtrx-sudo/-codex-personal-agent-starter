# 幼儿园日常节奏提醒模板

## 目标
用于第二天入园、周一升旗、园服、玩具、路线、接送等短提醒。

## 适用场景
- 周一提醒
- 升旗仪式提醒
- 次日入园提醒
- 接送时间提醒
- 特定物品携带提醒

## 语料锚点
- 明天周一会进行升国旗仪式
- 请...前入园
- 背上书包带上作业
- 开启元气满满的一周

## 最少需要的信息
- 日期或明天的事项
- 入园或接送时间
- 需带物品
- 是否穿园服
- 是否有路线或通道说明

## 默认结构
1. 开场称呼
2. 明天事项提醒
3. 时间要求
4. 物品或着装要求
5. 简短收尾

## 高频表达锚点
- `爸爸妈妈们：`
- `明天周一`
- `请孩子们...前入园`
- `请穿好园服`
- `开启元气满满的一周`

## 输出骨架

```md
爸爸妈妈们：

明天周一有升旗仪式，请孩子们今晚早些休息哦😊

请宝贝们 8:15 前入园，穿好园服，带上小书包和水杯。如有需要，也可以提醒孩子带上老师要求的小物品。

让我们一起开启元气满满的一周吧🌷
```

## 审校重点
- 时间必须短句直给
- 当日消息密度要高，但不能写成长文
- 适合老师在群里一条直接发出

## 待确认项
- 具体入园时间
- 是否要写接送通道或路线
- 是否要加 `@所有人`
