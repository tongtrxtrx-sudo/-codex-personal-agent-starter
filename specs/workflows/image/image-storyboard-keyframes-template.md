# 分镜关键帧子场景模板

## 目标
用于九宫格分镜、多镜头关键帧和角色设定页。

## 最少需要的信息
- 故事主题
- 角色主体
- 画面数量
- 每个镜头的大意
- 风格方向
- 一致性要求

## 推荐补问模板
```text
按分镜关键帧场景处理，现在还缺最少这几项：

故事主题：
角色主体：
需要几格/几张：
每格大意：
想要的风格：
需要保持一致的元素：
```

## Prompt 骨架
```text
multi-frame storyboard, {{角色主体}}, {{镜头主题}}, consistent character design, clear frame variation, {{风格方向}}, readable layout
```

## Negative Prompt 片段
```text
inconsistent character design, repeated camera angle, unclear scene transitions, cluttered panel layout, weak storytelling
```

## 二轮修改模板
```text
保留角色设定不变。
请让每一格镜头差异更明显，同时保持服装、发型和脸部一致。
不要让九宫格只像重复截图。
```

## 审校重点
- 角色一致性
- 镜头差异性
- 每格是否讲清楚

## 待确认项
- 最终是九宫格还是多张独立图
- 是否需要留文字区
