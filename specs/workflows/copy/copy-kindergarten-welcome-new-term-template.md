# 幼儿园欢迎新学期模板

## 目标
用于新学期进群欢迎、老师自我介绍、班级氛围建立和基础群规则同步。

## 适用场景
- 开学前建群欢迎
- 升班欢迎
- 新学期第一次正式群消息
- 新老师或新同学加入说明

## 语料锚点
- 欢迎各位加入幼儿园这个大家庭
- 我是本班班主任某老师
- 班级像一个大家庭，需要我们共同维护
- 进群的家长请备注孩子名字 + 家长称呼 + 手机号

## 最少需要的信息
- 幼儿园或班级名称
- 老师称呼
- 是否需要介绍其他老师或新同学
- 是否有家长群备注格式
- 是否要附群规或基本沟通说明

## 默认结构
1. 开场称呼
2. 欢迎加入和老师介绍
3. 班级氛围或家园共育说明
4. 备注格式或群内动作要求
5. 温和收尾

## 高频表达锚点
- `亲爱的家长们：`
- `欢迎各位加入...这个大家庭`
- `为了方便促进家园沟通，实现家园共育`
- `班级就像一个大家庭，需要我们共同维护`
- `进群的家长请备注...哦`

## 输出骨架

```md
亲爱的家长们：

大家好！欢迎各位加入小一班这个大家庭😊 我是本班班主任王老师，班级还有李老师和陈老师，接下来将和大家一起陪伴孩子们快乐成长。

为了方便促进家园沟通，实现家园共育，我们建立了班级家长群。班级就像一个大家庭，也需要我们一起用心维护。

进群的家长请备注：孩子名字 + 爸爸 / 妈妈 + 手机号，方便后续联系和沟通哦🤝

期待接下来和大家一起见证宝贝们的成长，谢谢大家的配合与支持🌷
```

## 审校重点
- 像老师欢迎家长，不像学校公告
- 身份介绍和群动作要求清楚
- 温暖但不空泛

## 待确认项
- 班级名称
- 老师名单
- 是否要写群规
- 是否保留 `@所有人`
