# 图片方向输出

## 方向 A
- 方向名：主角压场感
- 适用原因：最适合海报主视觉，角色存在感最强，风险最低
- 视觉描述：原创电系萌宠位于画面中心，轮廓清楚，表情生动，自带主角光
- Prompt：Scene / Background: restrained atmospheric background with soft electric ambience. Subject: single character key visual, original electric cute creature with hero presence. Key details: yellow body, red accents, long ears, lightning tail, lively expression, premium cinematic illustration style, strong silhouette, clean rim light. Composition and use: poster-ready composition with clear focal center and breathing space for title. Constraints: no copyrighted character copying, no cheap game poster feel, no childish toy rendering, no cluttered background
- Negative Prompt：cheap mobile game splash art, childish toy look, plush toy appearance, overdecorated scene, weak subject focus
- 主标题建议：首轮不建议上字
- 风险提醒：如果电光太多，容易滑向廉价战斗海报

## 方向 B
- 方向名：明亮陪伴感
- 适用原因：更适合社媒封面和轻传播图
- 视觉描述：角色更亲切、更治愈，背景明亮但克制，整体更偏陪伴型主视觉
- Prompt：Scene / Background: bright clean background with soft energy accents. Subject: single character illustration, original electric cute creature. Key details: yellow body, red cheek accents, long ears, lightning tail, charming smile, premium cinematic mood, soft light, polished illustration quality. Composition and use: clean composition suitable for social cover and lightweight poster use. Constraints: no low-end kids illustration, no cheap rendering, no noisy visual effects
- Negative Prompt：overly childish cartoon, plastic figurine feel, cheap poster style, flashy electric effects, crowded background
- 主标题建议：适合后续轻量上字
- 风险提醒：如果太软，会削弱主角压场感

## 方向 C
- 方向名：收藏级手绘感
- 适用原因：更适合高质感角色封面和收藏图
- 视觉描述：强调体积、材质和手绘完成度，背景极简，角色像精修插画收藏图
- Prompt：Scene / Background: minimal elegant background. Subject: stylized original electric creature poster illustration. Key details: rounded but not childish proportions, yellow body, red accents, long ears, lightning tail, expressive face, handcrafted painted finish, tactile shading, premium collectible quality. Composition and use: high-end cover composition with strong character focus. Constraints: no plastic toy cheapness, no overdecorated scene, no broken character features
- Negative Prompt：plastic toy rendering, fake glossy texture, overbusy background, childish proportions, low-end merchandising look
- 主标题建议：适合高质感封面
- 风险提醒：如果背景太空，冲击力会不如方向 A
