# 幼儿园报名 / 缴费 / 回执模板

## 目标
用于报名、缴费、托管意愿回收、信息表填写和截图回执提醒。

## 适用场景
- 开学报名通知
- 缴费提醒
- 课后托管报名
- 保险参保说明
- 问卷或链接填写

## 语料锚点
- 明日报名活动安排如下
- 请每一位家长进入链接填写报名意愿
- 完成缴费以后记得截图
- 不愿参保或有其他原因的请私信我说明

## 最少需要的信息
- 事项名称
- 截止时间
- 操作方式
- 是否需要付款
- 是否需要截图、回执或私信

## 默认结构
1. 开场称呼
2. 事项背景或一句目的
3. 分点列动作步骤
4. 补充截图、备注、私信等要求
5. 明确截止时间

## 高频表达锚点
- `亲爱的家长朋友们：`
- `现开启...`
- `请每一位家长...`
- `完成后记得截图`
- `收到请回复`

## 输出骨架

```md
亲爱的家长朋友们：

为了方便后续安排，现开启课后托管报名，请家长们按要求完成填写哦😊

1⃣️请进入链接填写报名意愿  
2⃣️如需缴费，请按页面提示完成  
3⃣️完成后记得截图或回执发给老师

如暂时不报名，也麻烦简单回复一下，方便老师统计。谢谢大家的配合🌷
```

## 审校重点
- 动作路径清楚，一看就知道要做什么
- 截止时间和回执要求不能漏
- 付款、截图、备注规则要准确

## 待确认项
- 截止时间
- 链接或二维码
- 是否需要截图
- 是否要 `@所有人`
