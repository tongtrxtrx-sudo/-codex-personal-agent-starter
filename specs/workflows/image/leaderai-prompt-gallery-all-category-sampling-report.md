# Leader Prompt Gallery 全类别抽样报告

## 目标
覆盖 `Leader Prompt Gallery` 的全部 `16` 个分类，给出每类的样本量、模式特征和对当前仓库图片 workflow 的启发。

## 数据说明
- 数据来源：`catalog.json`
- 总条目数：`6644`
- 分类数：`16`
- 本报告不是 `6644` 条模板的逐条人工精读
- 本报告基于：
  - 分类统计
  - 关键词模式统计
  - 每类代表样本抽样观察

## 关键观察
- `05_人物肖像与写实摄影` 是绝对大头，说明人物、写真和电影感图是这个库的主流场景。
- `04_创意脑洞特效合成` 与 `03_创意广告品牌设计` 是最适合补强你当前海报 / KV / 概念图方向的两类。
- `09_故事分镜角色设定`、`10_卡通漫画电影角色`、`12_手工玩具手办` 最适合补强你当前的角色图链路。
- `11_图像编辑与修复增强` 和 `01_建筑室内空间设计` 明显提示了“参考图编辑 / 转换 / 保持结构不变”是高频任务，不应被当成边缘需求。

## 全类别概览

### 01_建筑室内空间设计
- 样本量：`230`
- 主要信号：
  - `transformation_task`：87
  - `reference_based`：52
  - `style_control`：50
- 代表样本：
  - `风格化制图：古风效果图【建筑、景观、规划均适用】`
  - `效果图洗图【建筑、室内、景观、规划均适用】`
- 结论：
  - 这一类最强的是“基于已有图纸或参考图做转换”
  - 结构里常见“保持视角 / 结构不变，再改风格或材质”
- 对仓库的启发：
  - `reference_edit` 子场景很有必要
  - “保持不变的部分”应该在 workflow 里更显式

### 02_电商产品虚拟摄影
- 样本量：`207`
- 主要信号：
  - `style_control`：71
  - `reference_based`：35

  - `纯白背景高端工业产品设计概念`
  - `电商详情图制作-餐饮海报`
- 结论：
  - 这一类偏产品静物、电商详情图、白底图和产品摄影

- 对仓库的启发：

  - 需要专门的产品摄影 negative prompt 词库

- 主要信号：
  - `structured_meta`：1637
  - `style_control`：801
  - `reference_based`：349
  - `negative_control`：243
- 代表样本：
  - `高还原度黑白暗调人像摄影`
  - `喧闹人群中静止清晰的电影感肖像`

- 结论：
  - 这是全站最强、最成熟的一类
  - prompt 高度聚焦：

    - 姿态
    - 表情
    - 光线
    - 镜头情绪
    - 皮肤与材质边界
- 对仓库的启发：
  - `写真类 negative prompt 词库` 应该优先补
  - 写真人像必须从“主体 + 光线 + 边界”出发，不能只堆风格词

## 综合结论

### 最值得当前仓库优先吸收的类别
1. `05_人物肖像与写实摄影`
2. `03_创意广告品牌设计`
3. `10_卡通漫画电影角色`
4. `11_图像编辑与修复增强`

### 最适合作为备用探索层的类别
1. `04_创意脑洞特效合成`
2. `07_艺术插画创意风格`
3. `12_手工玩具手办`

### 当前不必优先重度吸收的类别
1. `13_虚拟购物试穿试用`
2. `16_表情包趣味拼图`

## 对仓库的直接建议
- 保留你当前的主 workflow，不要被外部 prompt 库反向定义流程
- 继续把外部库的价值沉淀为：
  - `prompt 模式`
  - `negative prompt 片段`
  - `二轮修改模板`
- 下一轮优先补：
  1. `角色图 prompt 模板`
  2. `写真类 negative prompt 词库`
  3. `reference_edit` 子场景
