# Copy Workflow Minimal Intake

## Goal
Keep intake short enough for everyday use while still collecting the minimum information needed for a stable first draft.

## Rule
Ask only 3 to 5 questions first.  
If the answer is still insufficient for a reliable draft, ask only the missing blockers.  
Everything else should move to `human-check items`.

## Intake by Scene

### Activity Promo
- Time
- Deadline
- Price
- Audience
- Contact or signup entry

### Internal Notice
- Time arrangement
- Back-to-work or execution time
- Contact
- Required action

### Training Notice
- Topic
- Time
- Location
- Audience
- Contact

### Service Reminder
- Service name
- Deadline
- Target audience
- Price or free
- Booking entry

### Meeting Notice
- Topic
- Time
- Location or link
- Audience
- Contact or preparation notes

### Weekly Report
- Time range
- Completed work
- Current risks
- Next-step plan
- Audience

### Meeting Follow-up
- Meeting topic
- Key decisions
- Action items
- Owners
- Deadlines

### Hiring
- Role
- Location
- Core responsibilities
- Apply channel
- Platform

### Social Post
- Topic
- Audience
- Core message
- Style
- Platform

### Video Packaging
- Content topic
- Platform
- Core hook
- Tone
- CTA level

## Fallback Rule
If the scene is still unclear after the raw input:
1. Ask what kind of task it is.
2. Ask where it will be published.
3. Ask what outcome it is trying to achieve.
