# 参考图反推模板子场景模板

## 目标
用于用户直接给一张参考图时，把图里的结构、气质和风险拆成可复用 prompt 模板，而不是只做一次性的看图描述。

## 适用场景
- 想复刻一类图的感觉
- 想从参考图提炼可复用 prompt
- 想给 ChatGPT Pro 一份稳定模板
- 想从优秀图片中提炼 negative prompt 风险

## 不适用场景
- 想直接编辑原图，请改用 `reference_edit`
- 想完全复刻一模一样的构图和元素
- 需要品牌授权边界极严的商业执行图

## 最少需要的信息
- 参考图是什么
- 想复用这张图的哪类感觉
- 必须保留哪些视觉特征
- 哪些部分允许变化
- 最终用途

## 推荐补问模板
```text
按参考图反推模板场景处理，现在还缺最少这几项：

参考图是什么：
想复用这张图的哪类感觉：
必须保留哪些视觉特征：
哪些部分允许变化：
最终用途：
```

## 反推输出骨架
```text
Scene / Background:
{{从参考图提炼的场景和氛围}}

Subject:
{{主体类型和核心身份}}

Key details:
{{材质、光线、姿态、风格、配色、必须保留特征}}

Composition and use:
{{构图、比例、留白、用途}}

Constraints:
{{不要跑偏的方向、不要新增的元素、版权和品牌边界}}
```

## 负向风险提炼规则
- 如果参考图高级感主要靠背景克制，就加入 `no cluttered background`
- 如果参考图靠真实肤质或材质，就加入 `no plastic skin` 或 `no fake material texture`
- 如果参考图主体极强，就加入 `no weak subject focus`
- 如果参考图带文字但你只想复用气质，就加入 `no extra text`

## 二轮修改模板
```text
保留参考图的 {{风格核心}} 不变。
请把 {{允许变化部分}} 调整为 {{目标效果}}。
不要破坏 {{必须保留特征}}，也不要新增无关元素。
```

## 审校重点
- 提炼出来的是“这类图的模板”还是“这张图的描述”
- 必保留特征是否被抽对
- 是否提炼出了明确的风险压制词
- 是否足够支持下一轮生成

## 待确认项
- 是想复用这张图的结构，还是只复用气质
- 是否涉及版权角色、品牌包装或真人肖像
- 是否需要顺手生成一版相似图 prompt
