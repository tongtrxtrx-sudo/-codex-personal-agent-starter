# 图片品牌审校清单

## 方向层检查
- [ ] 方向之间足够区分
- [ ] 至少有一个方向明显适合当前受众
- [ ] 没有全部落入同一种风格

## 品牌层检查
- [ ] 符合品牌调性
- [ ] 不廉价
- [ ] 不过度卡通
- [ ] 不过度促销
- [ ] 不偏离行业预期

## 画面层检查
- [ ] 主体清楚
- [ ] 场景明确
- [ ] 画面不过挤
- [ ] 颜色不过乱
- [ ] 关键信息有位置

## 文案层检查
- [ ] 主标题可用
- [ ] 主标题不过长
- [ ] 主标题不空泛
- [ ] 必保留事实没有被删掉

## 风险层检查
- [ ] 没有虚构品牌规范
- [ ] 没有依赖高风险促销表达
- [ ] 没有把猜测写成强制要求
- [ ] 已明确需要人工确认的点
