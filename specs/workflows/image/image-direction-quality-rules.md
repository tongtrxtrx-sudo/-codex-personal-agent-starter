# 图片方向质量规则

## 目标
让图片 workflow 输出的方向和 prompt 更稳定，减少“方向同质化”“提示过空”“画面廉价化”这类常见问题。

## 方向层规则
1. 默认输出 3 个方向。
2. 3 个方向之间必须有明显区别，不能只是换几个形容词。
3. 每个方向都要能一句话说清“为什么适合这个场景”。

## Prompt 层规则
1. Prompt 必须同时包含主体、场景、风格、色彩或氛围。
2. Negative Prompt 不能只写泛泛的 “bad quality”，要压具体风险。
3. 不要把品牌假设写成硬性规范。

## 标题层规则
1. 海报主标题默认不超过 16 字。
2. 不要用空泛促销词代替活动主题。
3. 不要把过多事实都塞进标题。

## 风险层规则
1. 不要输出明显廉价促销感方向，除非 brief 明确要求。
2. 不要过度卡通化，除非受众和品牌明确允许。
3. 不要把画面做得过满，必须保留信息区空间。

## 审校优先级
1. 先看受众匹配
2. 再看品牌调性
3. 再看画面清晰度
4. 最后看标题和细节
