---
name: storyboard-reviewer
description: Use when a draft video script needs scene breakdown, publish copy checks, and final production-oriented review.
---

# Goal
Turn a draft script into a reviewable storyboard package and catch practical production issues before filming.

# Use When
- a spoken script already exists
- the team needs shot ideas, captions, and publish copy
- video output must be made more shootable and reviewable

# Do Not Use When
- the script itself is still unstable
- no video brief exists
- formal legal or platform review is required

# Input Requirements
- approved or near-approved script
- source brief
- platform expectations when known

# Output Requirements
Return:
- scene-by-scene storyboard
- cover title options
- publish caption
- comment CTA or interaction cue
- issues needing human review

# Execution Rules
1. Keep shots practical and easy to film.
2. Match each scene to the script's purpose.
3. Check whether CTA, caption, and on-screen facts remain aligned.
4. Flag abstract or hard-to-shoot scenes.
5. Keep publication review separate from filming ideas.

# Success Criteria
- the script becomes shootable
- supporting publish assets are coherent
- obvious production or messaging problems are surfaced early

# Risk Boundaries
- do not promise platform performance outcomes
- do not treat a reviewed storyboard as approved for posting
