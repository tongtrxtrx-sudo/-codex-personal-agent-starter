# 错误笔记

## 错误 - 仓库文档语言规则误判

- 创建日期：2026-03-23
- 更新日期：2026-03-23
- 项目：D:\work\myclaw\codex-personal-agent-starter
- 分支：main
- 提交：待下次提交更新

### 问题

- 在本仓库中新增 Markdown 文档时，误把多份仓库文档写成英文
- 同时把 `memory/profile.md` 中的代码注释偏好写成了“中文”
- 与仓库本地规则“仓库文档默认中文”不一致

### 发现

- 全局规则中的文档英文默认，不能覆盖本仓库更深层的 `AGENTS.md`
- 本仓库文档应优先使用中文
- 代码注释和脚本帮助文本可以单独保持英文，不应与文档语言混在一起

### 解决方案

- 一句话总结：仓库文档统一中文，代码注释和脚本帮助文本单独保持英文
- 先检查仓库根 `AGENTS.md`
- 再检查 `memory/profile.md`
- 新增或修改文档前，确认它是“仓库文档”还是“代码帮助文本”
- 如果是仓库文档，默认写中文
- 如果是代码注释或脚本帮助文本，默认写英文
