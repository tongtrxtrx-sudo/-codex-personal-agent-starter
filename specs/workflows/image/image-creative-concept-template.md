# 创意概念图子场景模板

## 目标
用于双重曝光、隐喻叠加、抽象概念图和脑洞特效图。

## 最少需要的信息
- 核心概念
- 主体 A
- 主体 B 或概念元素
- 使用场景
- 风格边界
- 明确禁区

## 推荐补问模板
```text
按创意概念图场景处理，现在还缺最少这几项：

核心概念：
主体A：
主体B/要融合的元素：
使用场景：
想要的风格：
不希望出现元素：
```

## Prompt 骨架
```text
concept-driven visual, {{主体A}} merged with {{主体B或概念元素}}, {{核心情绪}}, {{风格关键词}}, strong visual metaphor, clean readable composition, suitable for {{用途}}
```

## Negative Prompt 片段
```text
visual chaos, random symbolism, excessive effects, noisy composition, meaningless surrealism, cheap fantasy poster feel
```

## 二轮修改模板
```text
保留核心概念不变。
请让画面更集中、更清楚，减少无关特效和装饰。
不要让概念感压过可读性。
```

## 审校重点
- 概念是否一眼能读懂
- 是否华而不实
- 是否偏离品牌或受众

## 待确认项
- 是否只做探索方向
- 是否允许更激进的艺术化处理
