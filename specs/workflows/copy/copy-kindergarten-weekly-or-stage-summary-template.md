# 幼儿园周总结 / 阶段总结模板

## 目标
用于同步孩子们近期状态、表扬进步，并补充家长下一步配合点。

## 适用场景
- 开学第一天总结
- 入园第三天总结
- 一周小结
- 节点阶段总结

## 语料锚点
- 今天开学第一天大部分宝贝们都...
- 孩子们今天仍然有进步
- 晚上在家请多表扬
- 接下来我们可以这样助力孩子

## 最少需要的信息
- 孩子近期表现
- 具体进步点
- 家长需要继续配合的动作
- 是否同步照片或其他渠道信息

## 默认结构
1. 开场称呼
2. 同步整体状态
3. 表扬孩子和家长
4. 给出下一步配合建议
5. 温和收尾

## 高频表达锚点
- `各位家长朋友们：`
- `晚上好！`
- `孩子们都有不同方面的进步`
- `晚上在家请多表扬`
- `让我们一起接受明天的挑战吧`

## 输出骨架

```md
各位家长朋友们：

晚上好！今天宝贝们在幼儿园整体状态不错，能和小朋友一起游戏，也在慢慢适应新的节奏👍

今天孩子们有不少进步，也很感谢家长们这几天的信任、坚持和配合。晚上回家可以多表扬、多鼓励孩子，帮助他们把这份信心继续延续下去。

接下来也请大家继续配合：让孩子自己背书包和水杯，按时入园，如有特殊情况及时和老师沟通。让我们一起陪伴宝贝们慢慢适应、越来越棒😊
```

## 审校重点
- 总结要有具体观察，不要空夸
- 表扬孩子和家长要平衡
- 下一步配合项要短而明确

## 待确认项
- 是否同步照片或视频
- 是否要写第二天具体提醒
- 是否保留 `@所有人`
