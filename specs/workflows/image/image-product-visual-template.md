# 产品视觉子场景模板

## 目标
用于电商主图、产品海报、白底图和产品摄影概念图。

## 最少需要的信息
- 产品是什么
- 目标受众
- 用途
- 风格或品牌感
- 必须出现的细节
- 输出比例

## 推荐补问模板
```text
按产品视觉场景处理，现在还缺最少这几项：

产品主体：
目标受众：
使用场景：
想要的风格/品牌感：
必须出现细节：
输出比例：
```

## Prompt 骨架
```text
premium product visual of {{产品主体}}, {{场景或背景}}, {{品牌感}}, {{材质与光线}}, {{构图方式}}, suitable for {{用途与比例}}
```

## Negative Prompt 片段
```text
cheap e-commerce mockup, flat product lighting, unclear material texture, cluttered props, low-end catalog style, muddy reflections
```

## 二轮修改模板
```text
保留产品形状和品牌色不变。
请强化材质细节、光线层次和高级感，减少廉价电商图感觉。
不要增加杂乱道具。
```

## 审校重点
- 产品识别是否清楚
- 材质是否可信
- 是否适合白底 / 海报 / 场景图用途

## 待确认项
- 是否需要透明背景
- 是否需要上价格或卖点
