# Goal
Turn one approved video angle into a clear script that is easy to speak and easy to film.

# Use When
- a video angle has already been chosen
- the team needs a 30-60 second spoken script
- content production needs a usable draft before storyboard review

# Do Not Use When
- the hook and angle are still undecided
- the task is to edit shots or visuals only
- compliance review needs a specialist approver

# Input Requirements
- selected angle
- campaign brief
- must-keep facts such as dates, price, contact, and CTA

# Output Requirements
Return a script with:
- opening hook
- pain point or context
- offer or solution
- supporting details
- CTA

# Execution Rules
1. Keep spoken language natural.
2. Preserve mandatory facts exactly.
3. Limit major selling points to a small number.
4. End with a clear action.
5. Write for reading aloud, not for long-form prose.

# Success Criteria
- the script is short, clear, and speakable
- mandatory facts remain intact
- the CTA is easy to understand

# Risk Boundaries
- do not hide important terms in the script
- do not treat draft scripts as ready for public posting without review
