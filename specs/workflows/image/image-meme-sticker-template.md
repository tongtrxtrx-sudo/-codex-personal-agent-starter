# 趣味贴纸子场景模板

## 目标
用于表情包、贴纸、趣味拼图和角色表情差分图。

## 最少需要的信息
- 主体
- 风格方向
- 需要几张 / 几格
- 表情或动作范围
- 是否需要文字

## 推荐补问模板
```text
按趣味贴纸场景处理，现在还缺最少这几项：

主体：
风格方向：
需要几张/几格：
表情或动作范围：
是否需要文字：
```

## Prompt 骨架
```text
sticker or meme sheet, {{主体}}, {{风格方向}}, {{表情或动作范围}}, clean cutout-friendly composition, suitable for {{几格或用途}}
```

## Negative Prompt 片段
```text
messy sticker sheet, low readability, weak expression contrast, cluttered background, muddy outlines
```

## 审校重点
- 表情差异是否清楚
- 是否适合裁切成贴纸
- 是否偏低质趣味图

## 待确认项
- 是否透明背景
- 是否需要文字气泡
