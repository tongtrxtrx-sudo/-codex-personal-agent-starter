# 写真人像子场景模板

## 目标
用于单人 / 双人人像、海边写真、杂志感人像、情绪壁纸等真实人物图的方向探索。

## 最少需要的信息
- 人物主体
- 目标受众
- 使用场景
- 写真风格
- 光线或时间段
- 不希望出现的低质感问题

## 推荐补问模板
```text
按写真人像场景处理，现在还缺最少这几项：

人物主体：
目标受众：
使用场景：
想要的写真风格：
光线/时间段：
不希望出现元素：
```

## Prompt 骨架
```text
full-body or half-body portrait of {{人物描述}}, in {{真实场景}}, {{写真风格}}, {{自然光或电影光}}, {{表情与姿态}}, clean background control, premium editorial quality, suitable for {{用途}}
```

## Negative Prompt 片段
```text
cheap studio portrait style, plastic skin, over-smoothed beauty filter, tacky glamour pose, low-end catalog photo look, cluttered background
```

## 二轮修改模板
```text
保留人物设定和场景不变。
请让表情和姿态更自然，减少摆拍感和磨皮感。
不要做成廉价影楼样片风。
```

## 审校重点
- 人物主体是否清楚
- 光线是否服务气质
- 皮肤和材质是否自然
- 是否误入低俗、影楼或塑料质感

## 待确认项
- 是否需要明确服装
- 是否需要竖版或横版锁定
- 是否需要上文字
