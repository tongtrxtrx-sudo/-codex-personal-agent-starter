# 常用提示词

## 开工

```text
基于当前已知信息，用一段话告诉我今天最值得推进的事、第一步该做什么，以及是否有任何高风险动作需要我确认。
```

## 记录

```text
帮我记下这个想法，不要扩展，只要保留原意并整理成一段清楚的话：
```

## 拆解

```text
把这个模糊想法压成今天能做的 3 个具体动作，保持简短。
```

## 方案比较

```text
请用中文比较这几个方案的优缺点，并先给结论，再给理由。
```

## 需求澄清

```text
把这段需求整理成范围、约束、风险和下一步，不要写成长文。
```

## 晨报

```text
基于当前重点和最近记录，给我一个极简版工作简报。
```

## 决策辅助

```text
如果你必须给我一个当前最合理的建议，你会怎么建议？请先给结论。
```
