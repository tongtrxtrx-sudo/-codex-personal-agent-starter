# 社媒封面子场景模板

## 目标
用于社交媒体封面、栏目封面和吸睛传播图。

## 最少需要的信息
- 平台
- 主题
- 受众
- 风格方向
- 是否需要标题区

## 推荐补问模板
```text
按社媒封面场景处理，现在还缺最少这几项：

平台：
主题：
目标受众：
想要的风格：
是否需要标题区：
```

## Prompt 骨架
```text
social cover visual for {{平台}}, {{主题}}, {{风格方向}}, clean readable composition, clear title-safe area, suitable for social media cover use
```

## Negative Prompt 片段
```text
clickbait cheap cover, cluttered title area, weak focal point, noisy layout, overdecorated background
```

## 审校重点
- 标题区是否可用
- 是否适配平台封面习惯

## 待确认项
- 具体尺寸
- 是否需要 logo
