# 海报主视觉子场景模板

## 目标
用于活动海报、招生海报、品牌 KV 以及需要明确标题区和信息区的视觉图。

## 最少需要的信息
- 主体
- 受众
- 场景
- 品牌感
- 主标题或必须保留事实
- 版式比例

## 推荐补问模板
```text
按海报主视觉场景处理，现在还缺最少这几项：

画面主体：
目标受众：
使用场景：
想要的风格/品牌感：
主标题或必须保留事实：
版式比例：
```

## Prompt 骨架
```text
{{主体}}, {{场景}}, {{品牌感}}, {{风格关键词}}, {{必须出现元素}}, clear information area for headline and facts, poster-ready layout, suitable for {{比例}}
```

## Negative Prompt 片段
```text
cheap promo poster, tacky flyer look, cluttered text area, busy composition, overdecorated background, weak information hierarchy
```

## 二轮修改模板
```text
保留主体和配色不变。
请增加清楚的标题区和留白区，让画面更像可落地的海报主视觉。
不要把背景做得过满。
```

## 审校重点
- 是否有足够留白
- 必保留事实是否可上版
- 是否偏概念图而不是可用海报

## 待确认项
- 是否需要 logo
- 是否需要副标题
- 最终比例是否锁定
