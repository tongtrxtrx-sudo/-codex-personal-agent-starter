# 文案 Workflow 最省力调用模板

## 目标
让用户不必理解整套规则，只用一句可复制的话，就能更稳定地触发正确的文案 workflow。

## 最通用模板

```text
按文案 workflow 处理下面需求。
这是一个【场景】，
要发到【平台/渠道】，
目标是【目的】。
默认只输出最终推荐版本、备选版本、待确认项。
```

## 你只需要替换 3 个位置

### 1. 场景
常用写法：
- 内部通知
- 活动推广
- 培训通知
- 服务提醒
- 会议通知
- 工作汇报 / 周报总结
- 招聘文案
- 社交媒体内容帖
- 娱乐内容 / 视频简介

### 2. 平台 / 渠道
常用写法：
- 企业微信、邮件、短信
- 朋友圈、社群、短信
- 小红书
- 微博
- 抖音
- 知乎
- B站
- 微信公众号
- 视频号
- 快手

### 3. 目的
常用写法：
- 通知 / 同步
- 推广 / 转化
- 分享 / 表达
- 互动 / 传播
- 记录 / 包装

## 什么时候再展开链路

只有在以下情况，再补一句：

```text
展开链路
```

适用情况：
- 你想调试 workflow
- 你觉得结果不对
- 你要检查 brief / draft / review 哪一步出了问题
- 你在做模板优化

## 一句话记法

`先说这是什么事，再说发到哪，最后说想达到什么效果。`
