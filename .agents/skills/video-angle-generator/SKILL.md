description: Use when a campaign needs short-form video topics, hooks, and positioning before script writing starts.
---

# Goal
Generate a small set of strong video angles before writing a full script.

# Use When
- the team knows the campaign but not the story angle
- short-form video output feels repetitive or weak
- script writing is blocked by missing hooks

# Do Not Use When
- a chosen angle already exists
- the task is to review a finished script only
- the campaign brief still lacks core selling points

# Input Requirements
- campaign brief
- target platform or format
- audience pain points and selling points

# Output Requirements
Return 3 or more options with:
- title
- opening hook
- core selling point
- emotional tone

# Execution Rules
1. Make the first three seconds meaningful.
2. Keep each angle distinct.
3. Align hooks with audience pain points, not empty hype.
4. Keep claims supportable by the source brief.
5. Optimize for scriptability, not just catchy titles.

# Success Criteria
- at least one angle is obviously worth scripting
- hooks are usable and not generic
- the team can compare options quickly

# Risk Boundaries
- do not fabricate benefits or outcomes
- do not rely on fear, pressure, or manipulative claims by default
