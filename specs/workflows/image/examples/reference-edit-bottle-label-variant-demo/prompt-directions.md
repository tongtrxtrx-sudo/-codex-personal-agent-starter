# 图片方向输出

## 方向 A
- 方向名：轻改夏季版
- 适用原因：最稳，最适合“看起来还是同一个产品”
- 视觉描述：保留原包装结构和主色，只在标签局部增加浅蓝与柠檬清爽元素
- Prompt：Change only: the bottle label design. Keep exactly the same: bottle geometry, perspective, liquid color, lighting, background, logo position, and all core packaging structure. Target result: a summer-limited label with light blue accents, subtle lemon freshness cues, and a cleaner seasonal feel while still reading as the same product. Constraints: preserve realistic studio photography quality, no logo distortion, no text corruption, no pasted-on look, suitable for ecommerce detail and campaign poster use
- Negative Prompt：structural drift, logo distortion, text corruption, artificial blending, changed bottle shape, changed lighting
- 主标题建议：不适用
- 风险提醒：如果夏季元素太强，会让产品识别度下降

## 方向 B
- 方向名：清爽柠檬强化版
- 适用原因：更适合活动海报或夏季促销页
- 视觉描述：标签上增加更明确的柠檬和清凉视觉元素，但仍保持品牌主结构
- Prompt：Change only: the printed label artwork. Keep exactly the same: bottle shape, perspective, camera angle, liquid color, shelf-ready studio lighting, background, and logo integrity. Target result: a fresher summer-limited label with visible lemon cues, light blue highlights, and a cooling seasonal mood. Constraints: maintain realistic packaging print quality, keep typography legible, do not redesign the product, no added props or background changes
- Negative Prompt：packaging redesign, logo drift, incorrect text, perspective distortion, low-detail patching, added unrelated elements
- 主标题建议：不适用
- 风险提醒：更适合活动视觉，不一定最稳合电商主图

## 方向 C
- 方向名：高保真品牌延展版
- 适用原因：适合品牌更严的包装编辑场景
- 视觉描述：只做非常克制的季节化升级，更强调高保真和结构稳定
- Prompt：Change only: subtle seasonal details on the label. Keep exactly the same: bottle geometry, framing, lighting, shadows, background, logo, and core package typography. Target result: a premium summer-limited refinement with restrained cool-toned accents and minimal citrus cues. Constraints: high-fidelity preservation, no structural drift, no text changes unless required, no visual mismatch, suitable for close-up product use
- Negative Prompt：structural drift, text corruption, logo distortion, background mismatch, overdecorated seasonal graphics, artificial blending
- 主标题建议：不适用
- 风险提醒：季节感最弱，但保真度最高
