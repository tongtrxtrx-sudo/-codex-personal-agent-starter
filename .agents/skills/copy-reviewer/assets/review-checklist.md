## 事实核对
- [ ] 活动名称正确
- [ ] 日期正确
- [ ] 截止时间正确
- [ ] 价格正确
- [ ] 联系方式正确
- [ ] 地点或参与方式正确

## 风格核对
- [ ] 语气符合目标人群
- [ ] 不夸张
- [ ] 不制造焦虑
- [ ] 不使用无法证明的承诺
- [ ] 不偏离品牌口径

## 渠道核对
- [ ] 海报标题长度可用
- [ ] 海报标题保留了活动主题或核心利益点
- [ ] 海报标题 3 个版本职责清楚，不是同义改写
- [ ] 朋友圈文案适合公开传播
- [ ] 朋友圈文案保留了时间、截止日期、价格、联系方式或报名入口
- [ ] 朋友圈文案默认推荐版本清楚
- [ ] 社群通知信息完整
- [ ] 社群通知保留了对象、时间、核心动作和报名信息
- [ ] 短信长度可控
- [ ] 短信保留了最关键的时间、动作和联系方式
- [ ] 内部通知类内容没有遗漏返岗时间或动作要求
- [ ] 邮件通知类内容没有遗漏联系人或例外说明
- [ ] 如果是社交媒体内容，输出结构符合目标平台
- [ ] 如果是小红书/知乎/B站等内容平台，标题和正文关系清楚
- [ ] 如果是抖音/微博等短内容平台，信息密度和节奏适配
- [ ] 如果是公众号/视频号/快手，输出结构符合平台习惯
- [ ] 如果是工作汇报或会议纪要，结构清楚且便于执行

## 风险核对
- [ ] 没有遗漏必须包含的信息
- [ ] 没有编造事实
- [ ] 没有需要审批却未标出的内容
- [ ] 已明确哪些地方需要人工确认
